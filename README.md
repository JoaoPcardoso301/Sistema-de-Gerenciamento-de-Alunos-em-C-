# Sistema de Cadastro e Avaliação de Alunos

## Sobre o projeto
Esse programa foi feito em C++.
Permite cadastrar alunos e analisar o desempenho deles com base nas notas.

## O que o programa faz
- Cadastra alunos (nome, sexo, idade e notas).
- Calcula a média das notas.
- Informa se o aluno foi aprovado ou reprovado.
- Mostra dados da turma, como:
  - Quantidade de alunos
  - Percentual de aprovados e reprovados
  - Média de idade dos alunos

## Reguisitos
- Notas: de 0 a 10  
- Idade: de 0 a 90  
- Média de aprovação: definida pelo usuário  
- Sexo: M ou F
